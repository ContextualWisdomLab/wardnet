# Figma Commercial Readiness Frame Brief

Do not use Figma Code Connect for this project.

## Frames

1. Admin Overview
   - Header: product name, health status, persistence mode, DNSBL origin.
   - Grid: routes, threats, DNSBL entries, SOC KPIs, recent events.
2. Commercial Readiness
   - Readiness level, target value, pass/fail checks, blockers.
   - Deployment assets and buyer evidence paths.
3. Threat Feed Operations
   - Feed list, last update, TTL, imported threat count, imported DNSBL count.
4. Support Bundle
   - Health, KPIs, license metadata, readiness, evidence counts.

## Design Constraints

- Use compact operational density.
- Avoid nested cards.
- Use status badges and tables instead of decorative panels.
- Keep all labels short enough for mobile widths.
- Provide hover tooltips for unfamiliar icons.
